About

This collection demonstrates advanced 3D rendering techniques used in modern games, implemented using pure WebGPU API without any external libraries. Each example is self-contained and focuses on a specific rendering concept.

Requirements: A browser with WebGPU support (Chrome 113+, Edge 113+, or Safari Technology Preview)

01. Phong Lighting

Basic 3D rendering with ambient, diffuse, and specular lighting. Demonstrates vertex buffers, uniforms, and basic shading.

Basic

02. Shadow Mapping

Real-time shadow rendering using shadow maps. Shows multi-pass rendering and depth textures.

Intermediate

03. Normal Mapping

Surface detail enhancement using normal maps. Demonstrates tangent space calculations and texture sampling.

Intermediate

04. Skybox

Environment rendering using cube maps. Shows how to create immersive backgrounds for 3D scenes.

Basic

05. Particle System

GPU-accelerated particle effects using compute shaders. Demonstrates compute pipeline for game effects.

Advanced

06. Bloom Post-Processing

Glow effect using multi-pass post-processing. Shows render-to-texture and Gaussian blur techniques.

Advanced

Learning Path

Recommended order: Start with Phong Lighting and Skybox to understand the basics, then move to Normal Mapping and Shadow Mapping for intermediate techniques. Finally, explore Particle Systems and Bloom for advanced GPU programming.
